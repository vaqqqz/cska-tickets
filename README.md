# 🔴 Спартак · Трекер билетов на ЦСКА

Веб-приложение, которое автоматически отслеживает появление билетов на матч **ЦСКА** на сайте [tickets.spartak.com](https://tickets.spartak.com/matches?team=94974f94-27da-4350-81b3-9eb7afa82237) и отображает актуальный статус на собственной странице.

---

## 🗂 Структура проекта

```
spartak-tracker/
├── app.py               # Flask-сервер + фоновый мониторинг
├── requirements.txt     # Python-зависимости
├── templates/
│   └── index.html       # HTML-страница с таймером и статусом
├── static/
│   └── style.css        # Дизайн (чёрный фон, красные акценты)
└── README.md
```

## ⚙️ Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/spartak-tracker.git
cd spartak-tracker
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Запустить приложение

```bash
python app.py
```

Откройте браузер и перейдите по адресу: **http://127.0.0.1:5000**

---

## 🔍 Как работает мониторинг

1. При запуске `app.py` стартует **фоновый поток** (`threading.Thread`).
2. Поток каждые **45 секунд** (настраивается через `CHECK_INTERVAL`) загружает страницу Спартака с помощью `requests`.
3. HTML-ответ парсится через `BeautifulSoup`, извлекается весь текст.
4. Проверяется наличие слова **«ЦСКА»** (без учёта регистра).
5. Результат сохраняется в словарь `state` (защищён `threading.Lock`).
6. Фронтенд каждые 45 секунд делает запрос к `/api/status` и обновляет страницу.

```
[Браузер] ←── GET /          ←── Flask (index.html)
[Браузер] ←── GET /api/status ←── Flask (JSON статус)
                                        ↑
                               [Фоновый поток]
                                   ↓ каждые 45 сек
                          [tickets.spartak.com]
```

## 🎛 Настройки

В начале файла `app.py` можно изменить:

| Переменная       | По умолчанию | Описание                             |
|------------------|-------------|--------------------------------------|
| `CHECK_INTERVAL` | `45`        | Интервал проверки в секундах         |
| `KEYWORD`        | `"ЦСКА"`    | Искомое слово на странице            |
| `TARGET_URL`     | см. код     | URL для мониторинга                  |

## 🖥 Интерфейс

- **Чёрный фон**, белый текст, красные акценты — брутальный минимализм.
- Крупный заголовок **«ЕСТЬ ЛИ БИЛЕТЫ НА ЦСКА?»**
- Статус обновляется без перезагрузки страницы (через Fetch API).
- **Таймер обратного отсчёта** до 6 мая 20:00 (локальное время).
- Ссылка на страницу билетов для быстрого перехода.

---

## ⚠️ Важные замечания

- Сайт `tickets.spartak.com` может использовать JavaScript-рендеринг. Если `requests` не видит нужный контент — замените парсинг на **Playwright** или **Selenium**.
- Слишком частые запросы могут привести к блокировке IP. Рекомендуемый интервал: **30–60 секунд**.
- Приложение не хранит данные — состояние сбрасывается при перезапуске.
